# توثيق مشروع كتالوج شركة أرض الشرق

## نظرة عامة
كتالوج إلكتروني لعرض منتجات شركة أرض الشرق لتجارة المواد الغذائية المجمدة والمبردة في الأردن.

---

## 1. نظام التصميم (Design System)

### الألوان
- **الأساسي (Primary):** أزرق بارد `HSL(205, 85%, 45%)` — يعكس طابع المجمدات
- **الثانوي (Accent):** أزرق مائي `HSL(195, 80%, 40%)`
- **الخلفية:** رمادي فاتح بارد `HSL(210, 30%, 98%)`
- **ألوان الفئات:** لكل فئة منتجات لون مخصص (أجبان: ذهبي، لحوم: أحمر، أسماك: أزرق، خضروات: أخضر...)

### الخطوط
- **Cairo** — خط عربي احترافي من Google Fonts

### التأثيرات
- `gradient-hero`: تدرج أزرق للهيدر والفوتر
- `gradient-frost`: تدرج ثلجي خفيف
- `shadow-card` / `shadow-card-hover`: ظلال ناعمة للبطاقات
- `animate-fade-in-up` / `animate-scale-in`: حركات ظهور سلسة

### الملفات المعدّلة
- `src/index.css` — متغيرات CSS المخصصة + اتجاه RTL
- `tailwind.config.ts` — ألوان، خطوط، حركات مخصصة

## 4. الميزات المنفّذة

- ✅ تصميم عربي RTL بالكامل
- ✅ هيدر ترويجي مع صورة بانر
- ✅ عرض العلامات التجارية (نبيل، الوادي، أمريكانا، لورباك، بوك، الإكرام، تب توب)
- ✅ بحث فوري عن المنتجات
- ✅ فلترة حسب الفئة
- ✅ عرض عدد المنتجات
- ✅ تصنيف المنتجات مع عناوين وأيقونات
- ✅ بطاقات منتجات مع شارة العلامة التجارية
- ✅ تأثيرات حركية عند التحميل والتمرير
- ✅ ذيل صفحة مع معلومات الشركة
- ✅ تصميم متجاوب (موبايل + ديسكتوب)
